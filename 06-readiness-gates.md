# Readiness Gates

## Gate 1: Discovery Ready

- project classified as greenfield or brownfield
- platform scope known
- backend docs status known

## Gate 2: Wireframe Ready

- screen inventory exists
- top-priority flows identified
- current state audited if brownfield

## Gate 3: Prototype Ready

- wireframes reviewed
- navigation model stable enough to test
- desktop/mobile hierarchy clear

## Gate 4: Contract Ready

- wireframe accepted
- prototype logic accepted
- data contract source known
- backend sample values available or clearly marked assumed

## Gate 5: Implementation Ready

- page contract frozen
- files/components likely to change identified
- open product questions resolved or explicitly deferred
